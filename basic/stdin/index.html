    標準入力¶

    よく使う標準入力¶

    文字列¶

    string

    S = input()

    数字¶

    1

    N = int(input())

    複数の数字¶

    別々の変数¶

    10 5

    N, K = map(int, input().split())

    1つの変数（リスト）¶

    1 2 3 4 5

    A = [int(x) for x in input().split()]

    複数行の文字列¶

    #..#
..##

    S = [list(input()) for _ in range(H)]

    Example

    >>> print(S)
[['#', '.', '.', '#'], ['.', '.', '#', '#']]

    複数行の数字¶

    1
2
3

    A = [int(input()) for _ in range(N)]

    複数行に複数の数字¶

    1 4
2 5
3 6

    1つの2次元リスト¶

    A = [[int(x) for x in input().split()] for _ in range(H)]

    Example

    >>> print(A)
[[1, 4], [2, 5], [3, 6]]

    別々の1次元リスト¶

    a, b = [None] * N, [None] * N
for i in range(N):
    a[i], b[i] = map(int, input().split())

    Example

    >>> print(a)
[1, 2, 3]
>>> print(b)
[4, 5, 6]

    標準入力の高速化¶

    import sys
input = sys.stdin.readline

    Note

    10倍以上高速化できるので、10^6 を超える場合は使うこと
